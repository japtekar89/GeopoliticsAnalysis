New Data from the Demographics and Financial Data API

The GDP API provides information on past and current GDP information for countries around the world. The GDP API offers a wide variety of GDP data per country, such as the GDP growth rate as a percentage, GDP at purchasing power parity, and nominal GDP. However, for this project, we only decided to focus on the nominal GDP variable, called gdp_nominal. This variable is measured in billions of US dollars. We also extracted the country variable, which is stored as a 3 letter country code, and the year. It is important to keep in mind that the API did not provide the nominal GDP for every country and/or for every year. The gdp_nominal variable also stored the predicted nominal GDP for future years, so we had to clean the data to remove years beyond 2025.

Gathering GDP Data from API

get_gdp <- function(country, key = my_key) {
  url <- glue("https://api.api-ninjas.com/v1/gdp?country={country};X-Api-Key={key}")
  my_data <- safely(fromJSON)(url)
  result <- my_data$result
  if (is.null(result) || length(result$year) == 0) {
    return(NULL)
  } else {
    return(
      data.frame(
        country = country,
        year = as.integer(unlist(my_data$result$year)),
        gdp_nominal = as.numeric(unlist(my_data$result$gdp_nominal))
      )
    )
  }
}

New Data from World Bank

The World Bank’s SDG database is a subset of its broader World Development Indicators, organized specifically around the UN’s 17 Sustainable Development Goals. It provides country-year panel data (typically yearly) covering economic, social, environmental, and institutional statistics aligned with each goal. The SDGs are separated by topic (e.g., poverty, education, gender equality).

The proportion of women in national parliaments variable is under the 5th SDG. The 5th SDG is “Achieve gender equality and empower all women and girls.” This variable measures the percentage of seats held by women in a country’s national parliament, typically focusing on the lower or single legislative chamber. It is reported annually and serves as an indicator of gender equality in politics.

This proportion is available for most countries from 1997-2020. There are missing values for certain years and countries, but the proportion is more available during the later years of that range.

As another test, let us check whether these results are statistically significant. We can run a two-sample t-test to compare the means of the results, seeing whether this difference is statistically significant.

t.test(prop_in_par ~ is_democracy, data = democracy_data)

Since the confidence interval does not include zero, we reject the null hypothesis and conclude that there is a statistically significant difference in the mean proportion of seats held by women between democratic and non-democratic countries. This suggests that the true difference in means is not zero.

The t-statistic is approximately −4.59 with 3382.5 degrees of freedom, and the p-value is very small, so it is less than a standard significance level of 0.05. This provides strong evidence against the null hypothesis and supports the conclusion that the difference in means is statistically significant.
